# 재작성 개념 노트

## 주제

주가 경로 시뮬레이션 (Stochastic Models & Market Risk)

## 직관적 요약

GBM 경로를 생성하고 terminal 분포, drawdown, barrier touch 확률을 요약한다. 핵심은 입력 가정과 계산 절차를 분리해, 결과가 어떤 가정에서 나왔는지 추적 가능하게 만드는 것입니다.

## 학습 순서

1. 먼저 샘플 입력 데이터의 열 정의를 확인합니다.
2. `src/`의 baseline 계산 함수를 실행합니다.
3. 결과 CSV에서 핵심 지표가 어떤 방향으로 움직였는지 확인합니다.
4. 실제 데이터나 더 큰 실험으로 확장하기 전에 한계와 누락 가정을 문서화합니다.
