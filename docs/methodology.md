# 방법론 노트

## 1. 프로젝트 목적

GBM 경로를 생성하고 terminal 분포, drawdown, barrier touch 확률을 요약한다.

## 2. 핵심 질문

동일한 기대수익률에서도 변동성 또는 기간이 바뀌면 경로 리스크는 얼마나 커지는가?

## 3. 입력 데이터

- 파일: `data/sample/gbm_path_assumptions.csv`
- 성격: 강의자료나 외부 원천을 복제하지 않은 합성 샘플
- 목적: 계산 흐름과 결과 해석을 작은 표본으로 먼저 검증

## 4. 계산 절차

1. GBM 이산화로 전체 경로를 만들고 terminal 가격과 경로 중 최저점을 기록한다.
2. 하방 barrier 터치 확률과 평균 최대낙폭을 함께 보아 단순 terminal 분석의 한계를 보완한다.
3. 모든 난수는 고정 seed와 합성 가정으로 생성한다.

## 5. 산출물 해석

`outputs/tables/baseline_results.csv`는 프로젝트별 baseline 결과입니다. 이 표는 최종 연구 결론이 아니라, 이후 실제 데이터 연결과 상세 분석을 시작하기 위한 구조 검증용 산출물입니다.

## 6. 한계

- 합성 데이터 기반이므로 통계적 유의성이나 투자 가능성을 주장하지 않습니다.
- 거래비용, 세금, 슬리피지, 유동성 제약은 일부 프로젝트에서만 단순 반영했습니다.
- 프로젝트별 정교한 모델링은 별도 후속 작업에서 확장해야 합니다.
